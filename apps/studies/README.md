# Studies Dashboard

Purpose: show module study artifacts, public pages, and selected CSV/JSON
previews.

Launch path:

```text
urlpath=/panel/studies_dashboard
```

The dashboard filters manifest entries by module, cluster, role, evidence type,
artifact format, and date. It also provides module tabs for DataLoader, DDP,
NCCL, GDS, HPL-MxP, and HPCG. Select an artifact and click the preview button
to fetch a small public CSV/JSON artifact from OSN, GitHub, or a bundled
generated CSV table.

Module notebooks are provided for DataLoader, DDP, GDS, HPCG, HPL-MxP, and
NCCL. Each one launches the same study dashboard focused to that module, so a
reviewer can start from either the combined dashboard or a specific module
lens. HPL-MxP FP8/FP4 and HPCG remain clearly marked as not-final dashboard
evidence until the public result story is updated.
